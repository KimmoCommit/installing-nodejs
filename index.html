Setting up GRUNT

Create package.json file to your project root. The package.json file has your project's module dependencies. Initially you might want to have the package.json file to have this kind of data:

{ "name": "example-project", "version": "0.1.0", "devDependencies": { "grunt": "~0.4.1" } }

Install some modules

Now you can find modules that you want to use in your development. Installation happens like so:

npm install grunt-sass --save-dev
npm install grunt-contrib-watch --save-dev

Configure your modules

Create Gruntfile.js to your project root and include your installed module configurations

module.exports = function(grunt) { grunt.initConfig({ pkg: grunt.file.readJSON('package.json'), sass: { options: { outputStyle: 'nested' }, dist: { files: { 'build/style.css': 'styles/SASS_sheet.scss' } } }, watch: { options: { livereload: true, }, css: { files: ['**/*.scss'], tasks: ['sass'], options: { spawn: false, } } } }); grunt.loadNpmTasks('grunt-sass'); grunt.loadNpmTasks('grunt-contrib-watch'); grunt.registerTask('default', ['sass'], ['watch']); };

Run GRUNT on project root

Now you can run your modules by opening command line prompt at the project root and running just:

grunt
